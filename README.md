# 🎓 AI Smart Classroom System

A complete AI-powered web application built with Python and Flask to automate and modernize classroom management.

## 🚀 Features

- 🔐 **Login System** — Secure login for Admin, Teacher and Student
- 📅 **Timetable Generator** — Manage class schedules
- 📸 **Attendance System** — Mark and track student attendance
- 🔔 **Smart Notifications** — Post announcements and topic updates
- 📊 **AI Performance Analytics** — Predict student performance
- 📝 **Notes Management** — Upload and download study materials
- 👥 **User Management** — Add and manage students and teachers

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| Python | Backend programming |
| Flask | Web framework |
| SQLite | Database |
| HTML + Bootstrap | Frontend UI |

## ⚙️ How to Run

1. Clone the repository
2. Create virtual environment
3. Install libraries with `pip install flask flask-sqlalchemy flask-login`
4. Run with `python app.py`
5. Open `http://127.0.0.1:5000`
